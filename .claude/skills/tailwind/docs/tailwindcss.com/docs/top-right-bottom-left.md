### [Basic example](https://tailwindcss.com/docs/top-right-bottom-left.html#basic-example)

Use `top-<number>`, `right-<number>`, `bottom-<number>`, `left-<number>`, and `inset-<number>` utilities like `top-0` and `bottom-4` to set the horizontal or vertical position of a [positioned element](https://tailwindcss.com/docs/position.html):


### [Using negative values](https://tailwindcss.com/docs/top-right-bottom-left.html#using-negative-values)

To use a negative top/right/bottom/left value, prefix the class name with a dash to convert it to a negative value:

```
<div class="relative size-32 ...">  <div class="absolute -top-4 -left-4 size-14 ..."></div></div>
```

Right-to-left

```
<div dir="ltr">  <div class="relative size-32 ...">    <div class="absolute start-0 top-0 size-14 ..."></div>  </div>  <div>    <div dir="rtl">      <div class="relative size-32 ...">        <div class="absolute start-0 top-0 size-14 ..."></div>      </div>      <div></div>    </div>  </div></div>
```

For more control, you can also use the [LTR and RTL modifiers](https://tailwindcss.com/docs/hover-focus-and-other-states.html#rtl-support) to conditionally apply specific styles depending on the current text direction.

### [Using a custom value](https://tailwindcss.com/docs/top-right-bottom-left.html#using-a-custom-value)

Use utilities like `inset-[<value>]` and `top-[<value>]` to set the position based on a completely custom value:

```
<div class="inset-[3px] ...">  <!-- ... --></div>
```

For CSS variables, you can also use the `inset-(<custom-property>)` syntax:

```
<div class="inset-(--my-position) ...">  <!-- ... --></div>
```

This is just a shorthand for `inset-[var(<custom-property>)]` that adds the `var()` function for you automatically.

### [Responsive design](https://tailwindcss.com/docs/top-right-bottom-left.html#responsive-design)

Prefix `inset`,`inset-x`,`inset-y`,`start`,`end`,`top`,`left`,`bottom`, and `right` utilities with a breakpoint variant like `md:` to only apply the utility at medium screen sizes and above:

```
<div class="top-4 md:top-6 ...">  <!-- ... --></div>
```

Learn more about using variants in the [variants documentation](https://tailwindcss.com/docs/hover-focus-and-other-states.html).

## [Customizing your theme](https://tailwindcss.com/docs/top-right-bottom-left.html#customizing-your-theme)

The `inset-<number>`,`inset-x-<number>`,`inset-y-<number>`,`start-<number>`,`end-<number>`,`top-<number>`,`left-<number>`,`bottom-<number>`, and `right-<number>` utilities are driven by the `--spacing` theme variable, which can be customized in your own theme:

```
@theme {  --spacing: 1px; }
```

Learn more about customizing the spacing scale in the [theme variable documentation](https://tailwindcss.com/docs/theme.html).
