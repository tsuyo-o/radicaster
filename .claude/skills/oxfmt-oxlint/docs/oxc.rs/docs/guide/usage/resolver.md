# Resolver [​](https://oxc.rs/docs/guide/usage/resolver.html#resolver)

* ![total downloads from crates.io](https://img.shields.io/crates/d/oxc_resolver)

Node.js cjs and esm path resolution.

## Features [​](https://oxc.rs/docs/guide/usage/resolver.html#features)

* All configurations are aligned with [webpack/enhanced-resolve](https://github.com/webpack/enhanced-resolve).
* 28x faster than [webpack/enhanced-resolve](https://github.com/webpack/enhanced-resolve) ([benchmark](https://github.com/oxc-project/bench-resolver)).
* See [README](https://github.com/oxc-project/oxc-resolver)
